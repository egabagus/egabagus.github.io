De Couff Cafe (Exploration)

UI and UX Design

In this exploration, I designed the UI and UX of a cafe profile website called De Couff. By using a lot of white space, I designed it with the aim of focusing the user on understanding the content on the website.

WIREFRAME
A wireframe below is a wireframe that I will use to design the UI for the cafe's website. The landing page of the De Couff website only displays menus, headings, CTAs and social menus in the form of a slider or carousel and a background of De Couff's photos.
ELEMENT
Using a white background color will make it easier for website visitors to easily understand the content inside. And I use the primary color brown, because it adjusts the color of the content on the website, which is coffee. The sans-serif typeface was chosen because it reflects the De Couff logo.
MOCKUP
In UI design, many use images or photos from the De Couff cafe. It is intended to tell the visitor what is in the shop and its history.
